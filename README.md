# 🎮 Tic-Tac-Toe AI Project

## 📌 Overview
This project implements a **Tic-Tac-Toe** game with **3 different AI algorithms**:
- 🎲 **Random Algorithm**
- ⚡ **Greedy Algorithm**  
- 🧠 **Minimax Algorithm**

## 🚀 Features
- Graphical User Interface (GUI) using Tkinter
- Choose between 3 AI algorithms
- Real-time game status display
- Reset game button
- Algorithm comparison


## 🛠️ Technologies Used
- Python 3.8+
- Tkinter (GUI)
- Random Library
- Minimax Algorithm

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/OSAMAFETEH/TicTacToe-AI-Project.git

# Navigate to project directory
cd TicTacToe-AI-Project

# Run the game
python main.py
